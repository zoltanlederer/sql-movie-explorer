# SQL Movie Explorer

> 🚧 Work in progress

CLI tool to query a personal movie library using SQLite and Python.

## Description

Takes the master CSV from Project 3 and loads it into a SQLite database.
Provides a CLI interface to filter by genre, year, rating, or type (movie vs TV show).
Results print cleanly in the terminal.

## Requirements

- Python 3.x
- pandas
